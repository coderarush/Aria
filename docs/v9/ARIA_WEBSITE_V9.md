## Blob Behavior System

The blob is Aria.

The blob is not decorative.

The blob is the main character of the website.

All motion should originate from the blob.

### Blob States

Idle

* Subtle breathing
* Minimal motion

Listening

* Slight expansion
* Gentle ripple

Thinking

* Organic morphing
* Slow transformation

Executing

* Motion radiates outward
* UI elements emerge

Complete

* Calm state
* Settled shape

Motion should feel:

* Organic
* Intelligent
* Calm
* Premium

Avoid:

* Bouncing
* Cartoon behavior
* Aggressive motion

---
